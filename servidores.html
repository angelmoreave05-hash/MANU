Tipos de Servidores Web

Apache

Servidor de código abierto muy flexible y ampliamente utilizado.

Nginx

Alta velocidad y eficiencia para manejar muchas conexiones.

IIS

Servidor de Microsoft enfocado en entornos Windows.

LiteSpeed

Alternativa rápida y optimizada a Apache.

Hardware

CPU

Procesa todas las solicitudes del servidor.

RAM

Permite atender múltiples usuarios simultáneamente.

Disco

Almacena archivos y datos del sitio web.

Red

Conecta el servidor a internet.

Software

Sistema Operativo

Linux o Windows Server.

Servidor Web

Apache, Nginx o IIS.

Lenguajes

PHP, JavaScript o Python.

Base de Datos

MySQL o PostgreSQL.

Forma de Trabajo

1. El usuario entra a una página web.

2. El navegador envía una solicitud.

3. El servidor procesa la petición.

4. Envía la respuesta al usuario.
